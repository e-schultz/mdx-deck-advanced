# Rendering Lists & Conditionals

---

## Rendering Lists

- Remember - you can pass any JavaScript value or object as a value to a Prop
- This includes Arrays 
- You can use Array methods like `map`, `filter`, `reduce` to transform your data into components.

---

## Rendering Lists

- `map` is used to transform one object to another 
- Can transform data into JSX 
- Can also reduce a list of items into a Component 
- Lists should specify a `key` attribute 

---

### Rendering Lists - Keys 

- Keys help React identify which items have changed, are added, or are removed. 
- Keys should be given to the elements inside the array to give the elements a stable identity.
- If your record has an `id` - that is the most common attribute to use.

---

## Rendering Lists

```jsx
const people = [{ id: 0, firstName: "Evan", lastName: "Schultz" }];
/* .... */
// inside of a component
 return (
      <div>
        {people.map(person => (
            <Person 
              key={person.id} 
              firstName={person.firstName} 
              lastName={person.lastName} />
          )
        )}
      </div>
    );
```

## Conditional Rendering 
- There are many options for conditionally rendering a component in React
- To render nothing at all, a component should return `null`

---

## Ternary Statements

- The most basic way to conditionally render a component 
- Must be used in place of an if block if you want a conditional in JSX

```javascript
// condition ? trueResult : falseResult
myName === 'Evan' ? 'Hello, Evan!' : 'Hello, World!'
```

---

```javascript Inline Ternary
const LoggedInComponent({ isLoggedIn }) => (
  <header>
    <h1>Welcome!</h1>
    { isLoggedIn ? <ProfileComponent /> : null }
  </header>
)
```

---

## Other options
- React is just JavaScript, so you can use any control flow block outside of JSX
- JSX and React Components can be assigned to variables and used later 

---

```javascript If Example
const LoggedInComponent({ isLoggedIn }) => {
  let childComponent = null
  if (isLoggedIn) {
    childComponent = <ProfileComponent />
  }
  return (
    <header>
      <h1>Welcome!</h1>
      { childComponent }
    </header>
  )
}
```

---

```javascript Switch Example
const LoggedInComponent({ loggedInStatus }) => {
  const childComponent = loggedInStatus => {
    switch (variant) {
      case "loggedIn":
        return <ProfileComponent />
      case "verifyAccount":
        return <CompleteSignUp />
      default:
        return null
    }
  }
  return (
    <header>
      <h1>Welcome!</h1>
      { childComponent }
    </header>
  )
}
```
