# Object - Basics

- Key / Value Pair
- Value can be any type - including other objects, arrays, functions, etc
- Keys can be added / removed after an Object has been created
- Can access keys dynamically

# Objects - Property Access

- `obj.title`
- `obj['title']`
- `obj[variable]`

---

export default Code;

```javascript
const task = {
  id: 104,
  complete: true,
  priority: "high",
  title: "Complete Slides",
  assignedTo: {
    firstName: "Evan",
    lastName: "Schultz"
  }
};

function getProperty(prop, obj) {
  return obj[prop];
}

console.log(task.complete);
console.log(task["title"]);
console.log(getProperty("priority", task));
```

export default Code;

```javascript Object - Listing Properties
const task = {
  id: 104,
  complete: true,
  priority: "high",
  title: "Complete Slides"
};

let keys = Object.keys(task);

for (let i = 0; i < keys.length; i++) {
  let key = keys[i];
  console.log(`task[${key}] = ${task[key]}`);
}

for (let key in task) {
  console.log(`task[${key}] = ${task[key]}`);
}
```

# Object Literals

---

## What are they?

- Variable name becomes property name
- Short hand for creating objects
- Can use computed key names

---

export default Code;

```javascript Object Literals - Old Way
function createTask(id, title, priority, complete) {
  return {
    id: id,
    title: title,
    priority: priority,
    complete: complete
  };
}
createTask(0, "New Task", "high", false);
```

---

export default Code;

```javascript Object Literals - Short Hand
function createTask(id, title, priority, complete, keyName, value) {
  return {
    id,
    title,
    priority,
    complete,
    [keyName]: value
  };
}
createTask(0, "New Task", "high", false, "archive", false);
// Object {complete: false, id: 0, priority: "high", title: "New Task"}
```
